# Generator-basic

A personal project generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Clone this repo, then inside of it, `npm link`
- In an empty directory run: `yo basic`
